# VietScript
An IDE that uses the Python-based programming language is almost entirely in Vietnamese.


How to use:
Step 1: Run the Python Code, you will see an IDE GUI
Step 2: Code like normal, but use the syntax in Vietnamese

Full syntax dictionary:
in_ra: print,
nhap: input,
neu: if,
nguoc: else,
khac: elif,
lap: for,
khi: while,
trong: in,
ham: def,
tra_ve: return,
dung: True,
sai: False,
nhap_vien: import,
va: and,
hoac: or,
la: is,
khong': not,
lop: class,
voi: with,
nhu: as
